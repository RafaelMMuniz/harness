# US-030: Render funnel visualization with conversion rates

**Priority:** 30
**Status:** passes:false (briefing generated during --fresh)

## What this story accomplishes

After this story lands, the Funnels page renders a visual bar chart showing progressive drop-off across funnel steps, with conversion rates, drop-off counts, and color coding for conversion health.

## Acceptance Criteria (VERBATIM from prd.json — DO NOT paraphrase)

- After funnel query returns, render a horizontal or vertical bar chart showing each step
- Each step bar shows: event name, user count, conversion rate as percentage (relative to step 1), and drop-off count from the previous step
- Bars visually decrease in size proportional to the conversion rate (funnel shape)
- Between each pair of steps, show the step-to-step conversion rate (e.g., 'Step 1 → Step 2: 65%') and the drop-off count
- Overall conversion rate displayed prominently (e.g., 'Overall: 12% of users completed all steps')
- Color coding: healthy conversion (>50%) in green, moderate (20-50%) in yellow, poor (<20%) in red
- Empty state when no users match the funnel criteria: 'No users completed Step 1 in this date range'
- Uses Recharts BarChart or a custom SVG/div-based funnel visualization
- Typecheck passes
- Verify in browser using dev-browser skill
- Funnel visualization tests pass: npx playwright test ui-funnels.spec.ts (visualization, conversion rate tests)

## Spec anchors (READ these sections if the briefing is unclear)
