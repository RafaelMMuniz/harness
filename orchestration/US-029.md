# US-029: Build funnel analysis page with step builder

**Priority:** 29
**Status:** passes:false (briefing generated during --fresh)

## What this story accomplishes

After this story lands, MiniPanel has a Funnels page with a step builder where growth managers can define 2-5 event steps, select a date range, and trigger a funnel analysis. The visualization is deferred to US-030.

## Acceptance Criteria (VERBATIM from prd.json — DO NOT paraphrase)

- New 'Funnels' route replaces the placeholder, accessible from sidebar with the Filter lucide-react icon
- Page heading: 'Funnel Analysis'
- Step builder section: user can add 2-5 event steps, each is a dropdown populated from GET /api/events/names
- Add Step button (disabled when 5 steps exist), Remove Step button on each step (disabled when 2 steps remain)
- Steps are numbered (Step 1, Step 2, etc.) and displayed vertically
- Date range selector with the same presets as the Trends page (Last 7 days, Last 30 days, Last 90 days, Custom)
- An 'Analyze' button triggers POST /api/funnels/query with the selected steps and date range
- Loading state shown while the API processes
- Uses shadcn/ui Select, Button, Card components
- Typecheck passes
- Verify in browser using dev-browser skill
- Funnel page step builder tests pass: npx playwright test ui-funnels.spec.ts (step builder tests)
